# QRPCA-HQF-Algorithm

Official implementation of the paper **"Color Image Denoising via Quaternion-Based Nonconvex Half-Quadratic Regularized Robust PCA"**.

This repository is the first official release of the implementation accompanying the paper and is intended to facilitate reproducible research. The source code, experimental settings, and examples are provided to help readers reproduce the results reported in the paper.

Some fundamental quaternion algorithms used in this repository were provided by Prof. Liao and are available on his homepage. The core algorithm, experimental design, implementation, and manuscript preparation were completed by the first author.
